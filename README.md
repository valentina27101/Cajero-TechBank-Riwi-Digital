# Cajero-TechBank-Riwi-Digital

## Descripción
Este proyecto consiste en el desarrollo de un cajero automático programado en Python.
El sistema permite a los usuarios interactuar mediante un menú para realizar diferentes 
operaciones bancarias básicas.

## Funcionalidades
- Autenticación con PIN

- Consultar saldo
- Retirar dinero
- Depositar dinero
- Historial de movimientos
- Cerrar sesión
